## Описание проекта
В данной лабораторной работе приложение **catty-reminders-app** было полностью переведено на работу в контейнеризированной среде **Docker**. Был реализован современный CI/CD конвейер на базе **GitHub Actions**, который автоматизирует процессы тестирования, сборки и деплоя.

### Основные изменения:
- **Docker:** Приложение упаковано в оптимизированный Docker-образ на базе `python:3.12-slim`.
- **CI (Continuous Integration):** Настроен воркфлоу для автоматического прогона тестов (API, UI, Unit) в среде GitHub Runners.
- **GitHub Packages (GHCR):** Реализована автоматическая сборка и публикация Docker-образов в реестр GitHub.
- **CD (Continuous Deployment):** Настроен автоматический деплой на удаленную виртуальную машину при создании **GitHub Release**.
- **SSH & FRP:** Доставка обновлений происходит через защищенный SSH-туннель (порт 3120), проброшенный через FRP.

## Как это работает

1. **Push в ветку lab3:**
   - Запускается воркфлоу `CI`.
   - В облаке GitHub устанавливаются зависимости и браузеры для Playwright.
   - Прогоняются тесты. В случае успеха собирается Docker-образ.
   - Имя образа принудительно переводится в нижний регистр для соблюдения стандартов Docker.

2. **Создание Release:**
   - Запускается воркфлоу `Deploy`.
   - Бот подключается к серверу по SSH (порт 3120).
   - Выполняется `docker pull` свежего образа из GHCR.
   - Перезапускается контейнер `catty-container`.

## Оптимизация и исправления
В ходе выполнения работы были решены следующие технические сложности:
- **Размер образа:** Установка браузеров вынесена из Docker-образа в среду выполнения тестов CI. Это уменьшило размер образа с 800МБ до 180МБ, что решило проблему тайм-аутов при деплое.
- **Права доступа:** На сервере настроено правило `visudo (NOPASSWD)`, позволяющее Docker-командам выполняться без интерактивного запроса пароля.
- **Структура тестов:** Добавлен файл `tests/__init__.py` и настроен `PYTHONPATH`, чтобы тесты корректно импортировали библиотеку `testlib` внутри GitHub Actions.
