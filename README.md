# Catty: Автоматизация деплоя (Лабораторная работа №1)

Проект реализует автоматическое развертывание приложения при каждом обновлении кода в ветке `lab1`.

## Архитектура решения

1. **GitHub Webhook** отправляет уведомление на `webhook.ladygin.course.prafdin.ru`.
2. **FRP** перенаправляет трафик на локальный порт 8080.
3. **Webhook Handler** запускает процесс деплоя в фоновом режиме.
4. **Deploy Script** выполняет `git pull`, имитирует сборку/тесты и перезапускает сервис.

## Структура проекта
| Файл | Описание |
| :--- | :--- |
| `webhook_handler.py` | Обработчик запросов (Flask, порт 8080) |
| `deploy.sh` | Сценарий автоматического развертывания |
| `build.sh`, `test.sh` | Скрипты имитации этапов сборки и тестирования |
| `.commit_hash` | Файл с хешем последнего деплоя для проверки |
